# Polymarket Anomaly & Arbitrage Scanner

Scans Polymarket binary (Yes/No) markets for pricing anomalies and arbitrage opportunities.

## Setup
```bash
pip install -r requirements.txt
```

## Usage
```bash
python polymarket_scanner.py
python polymarket_scanner.py --max-markets 1000
python polymarket_scanner.py --min-liquidity 500
```

## Run tests
```bash
python test_scanner.py
```
